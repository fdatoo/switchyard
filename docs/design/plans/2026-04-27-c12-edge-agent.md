# C12 — Edge Agent (`gohome-edge`) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `gohome-edge`, an optional supervisor binary that hosts Carport drivers on remote hosts and proxies their `Run` streams to a primary `gohomed` over mTLS, with on-disk event buffering for outage tolerance. No automation logic on the edge — it is a transport bridge with persistence.

**Architecture:** A `Transport` abstraction is factored out of `internal/carport/` (UDS impl preserved verbatim, new TLS impl added) so the same `Driver.Run` plumbing serves local subprocess drivers and edge-hosted drivers. The primary grows an internal CA, an `edge_*` SQLite registry projected from a new `EdgeEvent`, and an `EdgeService` Connect-RPC handling enrollment-token mint/redeem, server-streaming `WatchAssignment`, cert renewal/rotation/revocation, and read views. A new sibling repo `gohome-edge` (added as a git submodule at `GoHome/gohome-edge/`) ships one binary that imports `gohome` for shared protos, supervises driver subprocesses, encrypts an at-rest assignment cache (XChaCha20-Poly1305 keyed via HKDF from the edge mTLS private key), and bridges per-driver UDS↔TLS streams with an append-only ring buffer for offline events. Carport's wire grows two additive `v1alpha2` fields (`RunUp.edge_seq` + `RunDown.IngestAck`) negotiated via the existing `Handshake.protocol_version` hook reserved by C2 DR-1.

**Tech Stack:** Go 1.25 (existing), `connectrpc.com/connect` (existing — `EdgeService`), `google.golang.org/grpc` (existing — Carport TLS path), `crypto/x509` + `crypto/tls` + `crypto/ed25519` (stdlib — internal CA, mTLS), `golang.org/x/crypto/chacha20poly1305` + `golang.org/x/crypto/hkdf` (cache encryption — added as a direct dep), `github.com/spf13/cobra` (existing — CLI on both binaries), `github.com/charmbracelet/lipgloss` (existing — styled CLI output on both binaries), `modernc.org/sqlite` (existing — registry tables), stdlib `testing` (no testify in new code, matching house style).

**Depends on:** C2 (Carport wire — extends `protocol_version` hook), C4 (Pkl loader and `gohome.config.Module`-shaped imports the new `gohome.edge` Pkl module plugs into), C7 (Connect-RPC listener — adds the `EdgeService` handler alongside existing services), C9 (auth — internal CA encryption-at-rest pattern reused; `auth_enrollment_tokens` table generalized; UDS-only `system:local` principal pattern reused for `MintEnrollmentToken`). C12 does **not** depend on C8 (MCP), C10 (web), or C11 (HA import).

```go
const (
    protocolV1Alpha1 = "v1alpha1"
    protocolV1Alpha2 = "v1alpha2"
)

// negotiateProtocolVersion returns the version both ends will speak on
// this Run stream. C12 introduces v1alpha2 (adds RunUp.edge_seq +
// RunDown.IngestAck). We negotiate down: if the driver announces
// v1alpha2, we speak v1alpha2; if it announces anything else (including
// future versions we don't recognize), we speak v1alpha1 and never emit
// the new envelopes. Empty string is a hard error.
func negotiateProtocolVersion(announced string) (string, error) {
    switch announced {
    case "":
        return "", errors.New("handshake: protocol_version is required")
    case protocolV1Alpha2:
        return protocolV1Alpha2, nil
    default:
        // v1alpha1 OR forward-compat for future versions: speak v1alpha1.
        return protocolV1Alpha1, nil
    }
}
```

Wire the result into the `Instance` (or wherever per-connection state lives) so subsequent code can ask "does this connection speak v1alpha2?".

```go
// Package transport abstracts the wire substrate carrying Carport gRPC
// connections. Two implementations: UDS (local subprocess drivers) and
// TLS (remote edge-hosted drivers). The carport.Host calls AcceptDriver
// once per connection regardless of substrate; per-connection state
// (cert subject, peer address) is carried on the returned Conn.
package transport

import (
    "context"
    "net"
)

// Conn is an accepted driver connection. Implementations wrap a
// net.Conn (UDS) or a *tls.Conn with attached identity metadata.
type Conn interface {
    net.Conn
    // PeerIdentity returns a substrate-specific identity descriptor.
    // For UDS: peer credentials (uid, pid). For TLS: cert subject + edge
    // slug derived from cert. Used by the host's accept path to decide
    // whether to admit this connection.
    PeerIdentity() PeerIdentity
}

// PeerIdentity is sum-of-products: only one field is set, depending on
// substrate. Callers type-switch.
type PeerIdentity struct {
    UDS *UDSPeer
    TLS *TLSPeer
}

type UDSPeer struct {
    UID int
    PID int
    // HandshakeSecret is the per-spawn secret seeded into the driver
    // env; the host verifies it during gRPC Handshake. Carried out-of-
    // band so the host can correlate the OS-level peer with the
    // handshake-claimed instance.
    HandshakeSecret string
}

type TLSPeer struct {
    EdgeSlug   string  // resolved from cert serial via edge.Registry
    CertSerial string  // hex-encoded
    NotAfter   int64   // unix seconds
}

// Listener accepts inbound driver connections. Both UDS and TLS
// listeners implement this; the host is substrate-agnostic.
type Listener interface {
    Accept(ctx context.Context) (Conn, error)
    Close() error
    Addr() net.Addr
}

// Dialer is used by the carport supervisor to reach a freshly-spawned
// driver subprocess (UDS-only path). The TLS path inverts direction
// (edges dial in to the host's Listener), so TLS does not implement
// Dialer.
type Dialer interface {
    Dial(ctx context.Context, addr string) (Conn, error)
}
```

```go
package transport

import (
    "context"
    "fmt"
    "net"
    "os"
    "syscall"
)

// NewUDSListener creates a Unix-domain-socket listener with the given
// permission bits applied immediately after bind. Caller must Close()
// when done; Close() unlinks the socket file.
func NewUDSListener(path string, mode os.FileMode) (Listener, error) {
    // Pre-clean any stale socket from a previous unclean shutdown.
    if err := os.Remove(path); err != nil && !os.IsNotExist(err) {
        return nil, fmt.Errorf("remove stale socket: %w", err)
    }
    raw, err := net.Listen("unix", path)
    if err != nil {
        return nil, fmt.Errorf("listen unix %s: %w", path, err)
    }
    if err := os.Chmod(path, mode); err != nil {
        raw.Close()
        return nil, fmt.Errorf("chmod %o: %w", mode, err)
    }
    return &udsListener{l: raw.(*net.UnixListener), path: path}, nil
}

type udsListener struct {
    l    *net.UnixListener
    path string
}

func (u *udsListener) Accept(ctx context.Context) (Conn, error) {
    if d, ok := ctx.Deadline(); ok {
        u.l.SetDeadline(d)
    }
    raw, err := u.l.AcceptUnix()
    if err != nil {
        return nil, err
    }
    return wrapUDSConn(raw)
}

func (u *udsListener) Close() error {
    err := u.l.Close()
    _ = os.Remove(u.path)
    return err
}

func (u *udsListener) Addr() net.Addr { return u.l.Addr() }

// UDSDialer dials a Unix-domain socket.
type UDSDialer struct{}

func (UDSDialer) Dial(ctx context.Context, addr string) (Conn, error) {
    var d net.Dialer
    raw, err := d.DialContext(ctx, "unix", addr)
    if err != nil {
        return nil, fmt.Errorf("dial unix %s: %w", addr, err)
    }
    uc, ok := raw.(*net.UnixConn)
    if !ok {
        raw.Close()
        return nil, fmt.Errorf("dial: expected *net.UnixConn, got %T", raw)
    }
    return wrapUDSConn(uc)
}

type udsConn struct {
    *net.UnixConn
    peer UDSPeer
}

func (u *udsConn) PeerIdentity() PeerIdentity {
    return PeerIdentity{UDS: &u.peer}
}

func wrapUDSConn(c *net.UnixConn) (*udsConn, error) {
    raw, err := c.SyscallConn()
    if err != nil {
        return nil, fmt.Errorf("syscallconn: %w", err)
    }
    var ucred *syscall.Ucred
    var sockerr error
    if err := raw.Control(func(fd uintptr) {
        ucred, sockerr = syscall.GetsockoptUcred(int(fd), syscall.SOL_SOCKET, syscall.SO_PEERCRED)
    }); err != nil {
        return nil, fmt.Errorf("control: %w", err)
    }
    if sockerr != nil {
        return nil, fmt.Errorf("getsockopt SO_PEERCRED: %w", sockerr)
    }
    return &udsConn{
        UnixConn: c,
        peer: UDSPeer{
            UID: int(ucred.Uid),
            PID: int(ucred.Pid),
            // HandshakeSecret is set by the supervisor via SetHandshakeSecret.
        },
    }, nil
}

// SetHandshakeSecret is called by the carport supervisor immediately
// after spawning the driver subprocess and before passing the
// connection into the host. The secret is communicated to the driver
// via env var; the host correlates it during the gRPC Handshake.
func (u *udsConn) SetHandshakeSecret(s string) {
    u.peer.HandshakeSecret = s
}
```

```go
package transport

import (
    "context"
    "crypto/tls"
    "crypto/x509"
    "encoding/hex"
    "errors"
    "fmt"
    "net"
)

var ErrUnknownEdgeCert = errors.New("transport: unknown edge cert")
var ErrRevokedEdgeCert = errors.New("transport: revoked edge cert")

// VerifyEdgeFunc is supplied by the host. Given a peer cert serial
// (hex) and its parsed cert, returns the TLSPeer descriptor
// (slug, expiry, ...) or an error to reject the connection.
type VerifyEdgeFunc func(serial string, cert *x509.Certificate) (TLSPeer, error)

type TLSListenerConfig struct {
    Addr       string                 // host:port; ":7443" for the canonical default
    ServerCert tls.Certificate        // primary's server cert (signed by internal CA)
    ClientCAs  *x509.CertPool         // primary's internal CA cert
    VerifyEdge VerifyEdgeFunc         // post-handshake admission check
}

func NewTLSListener(cfg TLSListenerConfig) (Listener, error) {
    if cfg.VerifyEdge == nil {
        return nil, errors.New("TLSListenerConfig.VerifyEdge is required")
    }
    raw, err := net.Listen("tcp", cfg.Addr)
    if err != nil {
        return nil, fmt.Errorf("listen tcp %s: %w", cfg.Addr, err)
    }
    tlsCfg := &tls.Config{
        Certificates: []tls.Certificate{cfg.ServerCert},
        ClientAuth:   tls.VerifyClientCertIfGiven,
        ClientCAs:    cfg.ClientCAs,
        MinVersion:   tls.VersionTLS13,
    }
    return &tlsListener{
        raw:     raw,
        tlsCfg:  tlsCfg,
        verify:  cfg.VerifyEdge,
    }, nil
}

type tlsListener struct {
    raw    net.Listener
    tlsCfg *tls.Config
    verify VerifyEdgeFunc
}

func (t *tlsListener) Accept(ctx context.Context) (Conn, error) {
    raw, err := t.raw.Accept()
    if err != nil {
        return nil, err
    }
    tlsConn := tls.Server(raw, t.tlsCfg)
    if err := tlsConn.HandshakeContext(ctx); err != nil {
        tlsConn.Close()
        return nil, fmt.Errorf("tls handshake: %w", err)
    }
    state := tlsConn.ConnectionState()
    if len(state.PeerCertificates) == 0 {
        tlsConn.Close()
        return nil, errors.New("tls: peer presented no client cert (mTLS required for Carport)")
    }
    leaf := state.PeerCertificates[0]
    serial := hex.EncodeToString(leaf.SerialNumber.Bytes())
    peer, err := t.verify(serial, leaf)
    if err != nil {
        tlsConn.Close()
        return nil, fmt.Errorf("verify edge cert: %w", err)
    }
    return &tlsConnWrap{Conn: tlsConn, peer: peer}, nil
}

func (t *tlsListener) Close() error  { return t.raw.Close() }
func (t *tlsListener) Addr() net.Addr { return t.raw.Addr() }

type tlsConnWrap struct {
    *tls.Conn
    peer TLSPeer
}

func (t *tlsConnWrap) PeerIdentity() PeerIdentity {
    return PeerIdentity{TLS: &t.peer}
}

// TLSDialer is used by gohome-edge to dial the primary's TLS listener.
// Lives here (not in gohome-edge) so the primary's tests can exercise
// dial without a circular dep.
type TLSDialer struct {
    ServerName string          // primary endpoint hostname (matches server cert SAN)
    ClientCert tls.Certificate // edge's client cert + key
    RootCAs    *x509.CertPool  // primary's internal CA cert
}

func (d TLSDialer) Dial(ctx context.Context, addr string) (Conn, error) {
    cfg := &tls.Config{
        ServerName:   d.ServerName,
        Certificates: []tls.Certificate{d.ClientCert},
        RootCAs:      d.RootCAs,
        MinVersion:   tls.VersionTLS13,
    }
    raw := &net.Dialer{}
    rawConn, err := raw.DialContext(ctx, "tcp", addr)
    if err != nil {
        return nil, fmt.Errorf("dial tcp %s: %w", addr, err)
    }
    tlsConn := tls.Client(rawConn, cfg)
    if err := tlsConn.HandshakeContext(ctx); err != nil {
        rawConn.Close()
        return nil, fmt.Errorf("tls handshake: %w", err)
    }
    state := tlsConn.ConnectionState()
    if len(state.PeerCertificates) == 0 {
        tlsConn.Close()
        return nil, errors.New("tls: server presented no cert")
    }
    leaf := state.PeerCertificates[0]
    return &tlsConnWrap{
        Conn: tlsConn,
        peer: TLSPeer{
```

```go
// Package ca holds the gohomed-internal Certificate Authority used to
// issue mTLS certs to gohome-edge agents and to mint the primary's own
// Carport-listener server cert. The CA private key is wrapped at rest
// with the daemon's at-rest key (provided via the Wrapper interface;
// in production this is the same key C9's auth credentials are wrapped
// with).
//
// Lifetimes:
//   CA cert:    10 years (rotation is a wipe-and-re-pair-everything event)
//   Server cert: 1 year
//   Edge cert:   90 days (auto-renewed at <=30d remaining)
package ca

import (
    "context"
    "crypto/ecdsa"
    "crypto/elliptic"
    "crypto/rand"
    "crypto/x509"
    "crypto/x509/pkix"
    "encoding/pem"
    "errors"
    "fmt"
    "math/big"
    "time"
)

var ErrNoCA = errors.New("ca: no CA persisted yet")

type Store interface {
    Load(ctx context.Context) (PersistedCA, error)
    Save(ctx context.Context, p PersistedCA) error
}

type Wrapper interface {
    Wrap(plaintext []byte) ([]byte, error)
    Unwrap(ciphertext []byte) ([]byte, error)
}

type PersistedCA struct {
    CertPEM      string
    EncryptedKey []byte
    ExpiresAt    time.Time
}

type Options struct {
    Subject  pkix.Name
    Lifetime time.Duration  // default 10y if zero
}

type Authority struct {
    cert    *x509.Certificate
    privKey *ecdsa.PrivateKey
}

func (a *Authority) CACert() *x509.Certificate { return a.cert }

func Bootstrap(ctx context.Context, store Store, wrap Wrapper, opt Options) (*Authority, error) {
    p, err := store.Load(ctx)
    if err == nil {
        return loadFromPersisted(p, wrap)
    }
    if !errors.Is(err, ErrNoCA) {
        return nil, fmt.Errorf("ca: load: %w", err)
    }
    return generateAndPersist(ctx, store, wrap, opt)
}

func loadFromPersisted(p PersistedCA, wrap Wrapper) (*Authority, error) {
    block, _ := pem.Decode([]byte(p.CertPEM))
    if block == nil {
        return nil, errors.New("ca: persisted cert is not PEM")
    }
    cert, err := x509.ParseCertificate(block.Bytes)
    if err != nil {
        return nil, fmt.Errorf("ca: parse cert: %w", err)
    }
    keyDER, err := wrap.Unwrap(p.EncryptedKey)
    if err != nil {
        return nil, fmt.Errorf("ca: unwrap key: %w", err)
    }
    key, err := x509.ParseECPrivateKey(keyDER)
    if err != nil {
        return nil, fmt.Errorf("ca: parse key: %w", err)
    }
    return &Authority{cert: cert, privKey: key}, nil
}

func generateAndPersist(ctx context.Context, store Store, wrap Wrapper, opt Options) (*Authority, error) {
    if opt.Lifetime == 0 {
        opt.Lifetime = 10 * 365 * 24 * time.Hour
    }
    if opt.Subject.CommonName == "" {
        opt.Subject = pkix.Name{CommonName: "gohome internal CA"}
    }

    key, err := ecdsa.GenerateKey(elliptic.P256(), rand.Reader)
    if err != nil {
        return nil, fmt.Errorf("ca: generate key: %w", err)
    }
    serial, err := newSerial()
    if err != nil {
        return nil, err
    }
    now := time.Now()
    tmpl := &x509.Certificate{
        SerialNumber:          serial,
        Subject:               opt.Subject,
        NotBefore:             now.Add(-1 * time.Minute),
        NotAfter:              now.Add(opt.Lifetime),
        IsCA:                  true,
        KeyUsage:              x509.KeyUsageCertSign | x509.KeyUsageCRLSign | x509.KeyUsageDigitalSignature,
        BasicConstraintsValid: true,
    }
    der, err := x509.CreateCertificate(rand.Reader, tmpl, tmpl, &key.PublicKey, key)
    if err != nil {
        return nil, fmt.Errorf("ca: self-sign: %w", err)
    }
    cert, _ := x509.ParseCertificate(der)
    pemBytes := pem.EncodeToMemory(&pem.Block{Type: "CERTIFICATE", Bytes: der})

    keyDER, err := x509.MarshalECPrivateKey(key)
    if err != nil {
        return nil, fmt.Errorf("ca: marshal key: %w", err)
    }
    wrapped, err := wrap.Wrap(keyDER)
    if err != nil {
        return nil, fmt.Errorf("ca: wrap key: %w", err)
    }
    if err := store.Save(ctx, PersistedCA{
        CertPEM:      string(pemBytes),
        EncryptedKey: wrapped,
        ExpiresAt:    cert.NotAfter,
    }); err != nil {
        return nil, fmt.Errorf("ca: save: %w", err)
    }
    return &Authority{cert: cert, privKey: key}, nil
}

func (a *Authority) SignEdgeCert(csr *x509.CertificateRequest, slug string, lifetime time.Duration) (*x509.Certificate, error) {
    if csr.Subject.CommonName != slug {
        return nil, fmt.Errorf("ca: CSR CN %q does not match slug %q", csr.Subject.CommonName, slug)
    }
    if err := csr.CheckSignature(); err != nil {
        return nil, fmt.Errorf("ca: csr signature invalid: %w", err)
    }
    serial, err := newSerial()
    if err != nil {
        return nil, err
    }
    now := time.Now()
    tmpl := &x509.Certificate{
        SerialNumber: serial,
        Subject:      csr.Subject,
        NotBefore:    now.Add(-1 * time.Minute),
        NotAfter:     now.Add(lifetime),
        KeyUsage:     x509.KeyUsageDigitalSignature | x509.KeyUsageKeyEncipherment,
        ExtKeyUsage:  []x509.ExtKeyUsage{x509.ExtKeyUsageClientAuth},
    }
    der, err := x509.CreateCertificate(rand.Reader, tmpl, a.cert, csr.PublicKey, a.privKey)
    if err != nil {
        return nil, fmt.Errorf("ca: sign edge cert: %w", err)
    }
    return x509.ParseCertificate(der)
}

func (a *Authority) SignServerCert(commonName string, dnsNames []string, lifetime time.Duration) (*x509.Certificate, *ecdsa.PrivateKey, error) {
    key, err := ecdsa.GenerateKey(elliptic.P256(), rand.Reader)
    if err != nil {
        return nil, nil, fmt.Errorf("ca: gen server key: %w", err)
    }
    serial, err := newSerial()
    if err != nil {
        return nil, nil, err
    }
    now := time.Now()
    tmpl := &x509.Certificate{
        SerialNumber: serial,
        Subject:      pkix.Name{CommonName: commonName},
        NotBefore:    now.Add(-1 * time.Minute),
        NotAfter:     now.Add(lifetime),
        KeyUsage:     x509.KeyUsageDigitalSignature | x509.KeyUsageKeyEncipherment,
        ExtKeyUsage:  []x509.ExtKeyUsage{x509.ExtKeyUsageServerAuth},
        DNSNames:     dnsNames,
    }
    der, err := x509.CreateCertificate(rand.Reader, tmpl, a.cert, &key.PublicKey, a.privKey)
    if err != nil {
        return nil, nil, fmt.Errorf("ca: sign server: %w", err)
    }
    cert, _ := x509.ParseCertificate(der)
    return cert, key, nil
}

func newSerial() (*big.Int, error) {
    // 128-bit random serial — matches BR-style serial uniqueness expectation.
    max := new(big.Int).Lsh(big.NewInt(1), 128)
    return rand.Int(rand.Reader, max)
}
```

```go
import (
    "context"
    "fmt"
    "time"

    eventv1 "github.com/fynn-labs/gohome/gen/gohome/event/v1"

// Projector replays EdgeEvent payloads into the edge_* tables. Idempotent
// — the projector is replay-safe.
type Projector struct {
    s *Store
}

func NewProjector(s *Store) *Projector { return &Projector{s: s} }

func (p *Projector) Apply(ctx context.Context, e *eventv1.EdgeEvent) error {
    switch k := e.GetKind().(type) {
    case *eventv1.EdgeEvent_Enrolled:
        return p.s.UpsertEdgeAgent(ctx, EdgeAgent{
            Slug:     k.Enrolled.Slug,
            PairedAt: ptr(time.Now()),
        })
    case *eventv1.EdgeEvent_EnrollmentTokenMinted:
        // Tokens are inserted directly by the service handler (which
        // has the secret bytes); the event is audit-only. No-op here.
        return nil
    case *eventv1.EdgeEvent_EnrollmentTokenRedeemed:
        // Consumption is also done atomically in the service; audit-only.
        return nil
    case *eventv1.EdgeEvent_CertRotated:
        return p.s.MarkCertRevoked(ctx, k.CertRotated.OldSerial, "rotated")
    case *eventv1.EdgeEvent_Revoked:
        return p.s.MarkCertRevoked(ctx, k.Revoked.CertSerial, k.Revoked.Reason)
    case *eventv1.EdgeEvent_Connected:
        return p.s.UpdateConnectionState(ctx, k.Connected.Slug, +1, time.Now(), nil, "")
    case *eventv1.EdgeEvent_Disconnected:
        now := time.Now()
        return p.s.UpdateConnectionState(ctx, k.Disconnected.Slug, -1, time.Time{}, &now, k.Disconnected.Reason)
    case *eventv1.EdgeEvent_AssignmentSynced:
        return p.s.UpsertEdgeAgent(ctx, EdgeAgent{
            Slug:           k.AssignmentSynced.Slug,
            LastSyncedAt:   ptr(time.Now()),
            AssignmentHash: k.AssignmentSynced.AssignmentHash,
        })
    case *eventv1.EdgeEvent_BufferOverflowed:
        // Audit-only; no projection update.
        return nil
    default:
        return fmt.Errorf("projector: unknown EdgeEvent kind %T", k)
    }
}

func ptr[T any](v T) *T { return &v }
```

```go
    Drivers          []DriverInstance
    AssignmentHash   string
}

type Resolver struct{ cfg ConfigSource }

func NewResolver(c ConfigSource) *Resolver { return &Resolver{cfg: c} }

func (r *Resolver) Resolve(ctx context.Context, slug string) (Bundle, error) {
    driverSlugs, ok := r.cfg.EdgeAgentDrivers(slug)
    if !ok {
        return Bundle{}, ErrUnknownEdge
    }
    sortedSlugs := append([]string(nil), driverSlugs...)
    sort.Strings(sortedSlugs)
    drivers := make([]DriverInstance, 0, len(sortedSlugs))
    for _, ds := range sortedSlugs {
        d, ok := r.cfg.DriverInstance(ds)
        if !ok {
            return Bundle{}, fmt.Errorf("assignment: driver instance %q is assigned to edge %q but missing from config", ds, slug)
        }
        drivers = append(drivers, d)
    }
    bundle := Bundle{
        EdgeSlug:         slug,
        SyncedAt:         time.Now(),
        PrimaryCACertPEM: r.cfg.PrimaryCACertPEM(),
        Drivers:          drivers,
    }
    bundle.AssignmentHash = canonicalHash(bundle)
    return bundle, nil
}

// canonicalHash produces a deterministic sha256 of the bundle bytes,
// excluding SyncedAt (which is wall-clock and shouldn't affect "is
// this assignment the same?"). Drivers are already sorted by Resolve.
func canonicalHash(b Bundle) string {
    pb := ToProto(b)
    pb.SyncedAt = nil  // exclude wall-clock
    bytes, _ := proto.Marshal(pb)
    sum := sha256.Sum256(bytes)
    return hex.EncodeToString(sum[:])
}

// ToProto converts Bundle to the wire shape. Public so the service
// handler can use it directly.
func ToProto(b Bundle) *gohomev1.AssignmentBundle {
    drivers := make([]*gohomev1.DriverInstanceAssignment, 0, len(b.Drivers))
    for _, d := range b.Drivers {
        drivers = append(drivers, &gohomev1.DriverInstanceAssignment{
            Slug:           d.Slug,
            DriverBinary:   d.Binary,
            DriverVersion:  d.Version,
            ResolvedConfig: d.ResolvedConfig,
            Env:            d.Env,
        })
    }
    return &gohomev1.AssignmentBundle{
        EdgeSlug:           b.EdgeSlug,
        SyncedAt:           timestamppb.New(b.SyncedAt),
        PrimaryCaCertPem:   b.PrimaryCACertPEM,
        Drivers:            drivers,
        AssignmentHash:     b.AssignmentHash,
    }
}
```

`internal/buffer/segment.go`:

```go
// Package buffer manages per-driver on-disk ring buffers for events
// flowing through gohome-edge. A buffer is a directory of length-
// prefixed protobuf log segments + a cursor.json file.
package buffer

import (
    "encoding/binary"
    "errors"
    "fmt"
    "io"
    "os"

    "google.golang.org/protobuf/proto"

    edgev1 "github.com/fynn-labs/gohome/gen/gohome/edge/v1"
)

// Segment is one log file inside a buffer directory. Records are
// written as: uint32 little-endian length || protobuf bytes.
type Segment struct {
    f         *os.File
    writeOff  int64
}

// OpenSegment opens (or creates) the segment file and returns it
// positioned for append. On open it scans for the last valid record
// and truncates anything after it (crash-recovery).
func OpenSegment(path string) (*Segment, error) {
    f, err := os.OpenFile(path, os.O_RDWR|os.O_CREATE, 0o600)
    if err != nil {
        return nil, fmt.Errorf("open segment %s: %w", path, err)
    }
    end, err := scanForLastValidRecord(f)
    if err != nil {
        f.Close()
        return nil, fmt.Errorf("scan segment %s: %w", path, err)
    }
    if err := f.Truncate(end); err != nil {
        f.Close()
        return nil, fmt.Errorf("truncate segment %s: %w", path, err)
    }
    if _, err := f.Seek(end, 0); err != nil {
        f.Close()
        return nil, err
    }
    return &Segment{f: f, writeOff: end}, nil
}

func scanForLastValidRecord(f *os.File) (int64, error) {
    if _, err := f.Seek(0, 0); err != nil {
        return 0, err
    }
    var off int64
    buf := make([]byte, 4)
    for {
        if _, err := io.ReadFull(f, buf); err != nil {
            if errors.Is(err, io.EOF) || errors.Is(err, io.ErrUnexpectedEOF) {
                return off, nil
            }
            return 0, err
        }
        length := binary.LittleEndian.Uint32(buf)
        body := make([]byte, length)
        if _, err := io.ReadFull(f, body); err != nil {
            // Torn body — return offset before this record.
            return off, nil
        }
        if err := proto.Unmarshal(body, &edgev1.BufferedEnvelope{}); err != nil {
            return off, nil
        }
        off += 4 + int64(length)
    }
}

func (s *Segment) Append(e *edgev1.BufferedEnvelope) error {
    body, err := proto.Marshal(e)
    if err != nil {
        return fmt.Errorf("marshal: %w", err)
    }
    var hdr [4]byte
    binary.LittleEndian.PutUint32(hdr[:], uint32(len(body)))
    if _, err := s.f.Write(hdr[:]); err != nil {
        return err
    }
    if _, err := s.f.Write(body); err != nil {
        return err
    }
    s.writeOff += 4 + int64(len(body))
    return nil
}

func (s *Segment) Sync() error  { return s.f.Sync() }
func (s *Segment) Close() error { return s.f.Close() }
func (s *Segment) Size() int64  { return s.writeOff }

// Iterate returns an iterator that walks the segment from start.
func (s *Segment) Iterate() (*SegmentIter, error) {
    f, err := os.Open(s.f.Name())
    if err != nil {
        return nil, err
    }
    return &SegmentIter{f: f}, nil
}

type SegmentIter struct {
    f *os.File
}

func (it *SegmentIter) Next() (*edgev1.BufferedEnvelope, bool) {
    hdr := make([]byte, 4)
    if _, err := io.ReadFull(it.f, hdr); err != nil {
        return nil, false
    }
    length := binary.LittleEndian.Uint32(hdr)
    body := make([]byte, length)
    if _, err := io.ReadFull(it.f, body); err != nil {
        return nil, false
    }
    e := &edgev1.BufferedEnvelope{}
    if err := proto.Unmarshal(body, e); err != nil {
        return nil, false
    }
    return e, true
}

func (it *SegmentIter) Close() error { return it.f.Close() }
```

```go
// Package cache stores the last-known AssignmentBundle to disk,
// encrypted with XChaCha20-Poly1305 keyed via HKDF-SHA256 from the
// edge's mTLS private key. Decryption failure (e.g. after cert
// rotation regenerates the key) is treated as "no cache" by callers.
package cache

import (
    "crypto/ed25519"
    "crypto/rand"
    "errors"
    "fmt"
    "io"
    "os"

    "golang.org/x/crypto/chacha20poly1305"
    "golang.org/x/crypto/hkdf"
    "google.golang.org/protobuf/proto"

    gohomev1 "github.com/fynn-labs/gohome/gen/gohome/v1alpha1"
)

const hkdfInfo = "gohome-edge cache v1"

type Cache struct {
    path string
    key  ed25519.PrivateKey
}

func New(path string, key ed25519.PrivateKey) *Cache { return &Cache{path: path, key: key} }

func (c *Cache) Save(b *gohomev1.AssignmentBundle) error {
    pb, err := proto.Marshal(b)
    if err != nil {
        return fmt.Errorf("marshal: %w", err)
    }
    aead, err := newAEAD(c.key)
    if err != nil {
        return err
    }
    nonce := make([]byte, aead.NonceSize())
    if _, err := rand.Read(nonce); err != nil {
        return err
    }
    ct := aead.Seal(nil, nonce, pb, nil)
    payload := append(nonce, ct...)

    tmp := c.path + ".tmp"
    if err := os.WriteFile(tmp, payload, 0o600); err != nil {
        return err
    }
    return os.Rename(tmp, c.path)
}

func (c *Cache) Load() (*gohomev1.AssignmentBundle, error) {
    payload, err := os.ReadFile(c.path)
    if err != nil {
        return nil, err
    }
    aead, err := newAEAD(c.key)
    if err != nil {
        return nil, err
    }
    if len(payload) < aead.NonceSize() {
        return nil, errors.New("cache: payload too short")
    }
    nonce := payload[:aead.NonceSize()]
    ct := payload[aead.NonceSize():]
    pt, err := aead.Open(nil, nonce, ct, nil)
    if err != nil {
        return nil, fmt.Errorf("cache: decrypt: %w", err)
    }
    b := &gohomev1.AssignmentBundle{}
    if err := proto.Unmarshal(pt, b); err != nil {
        return nil, fmt.Errorf("cache: unmarshal: %w", err)
    }
    return b, nil
}

func newAEAD(privKey ed25519.PrivateKey) (chacha20poly1305.AEAD, error) {
    h := hkdf.New(sha256New, privKey.Seed(), nil, []byte(hkdfInfo))
    var key [chacha20poly1305.KeySize]byte
    if _, err := io.ReadFull(h, key[:]); err != nil {
        return nil, err
    }
    return chacha20poly1305.NewX(key[:])
}

// sha256New aliases stdlib to avoid two-line import noise.
var sha256New = func() hash.Hash { return sha256.New() }
```
