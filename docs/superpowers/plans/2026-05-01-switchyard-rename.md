# Switchyard Rename Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rename the project from GoHome to Switchyard across all module paths, binaries, proto packages, Pkl config, and documentation.

**Architecture:** Pure rename — no logic changes. Work proceeds in layers: filesystem renames → module files → proto + codegen → Go source sed pass → Pkl config → build files + docs → verify. Each layer unblocks the next.

**Tech Stack:** Go workspace (go.work), buf (proto codegen), Pkl (config schema), Taskfile

### Task 5: Bulk sed pass on Go source files

**Files:**
- Modify: all `*.go` files outside `gen/` (generated files were just regenerated correctly)

Three substitutions are needed, applied in order:
1. Module path: `github.com/fdatoo/gohome` → `github.com/fdatoo/switchyard`
2. Gen subdirectory: `/gen/gohome/` → `/gen/switchyard/`
3. Connect package name: `gohomev1alpha1connect` → `switchyardv1alpha1connect`

- [ ] **Step 1: Apply module path substitution to all non-generated Go files**

```bash
find . -name "*.go" -not -path "./gen/*" -not -path "./.git/*" -exec \
  sed -i '' 's|github\.com/fdatoo/gohome|github.com/fdatoo/switchyard|g' {} +
```

- [ ] **Step 2: Apply gen subdirectory substitution**
